Mesemu

A decent NES emulator written in Rust and powered by WebAssembly.

Features

  • Supports iNES 1.0 file format.
  • Supported mappers: NROM.
  • Implements some of the original hardware bugs and quirks.

Controls

  • Directions: ↑ ↓ ← →
  • A: Q / A
  • B: Z / W
  • Select: Space
  • Start: Enter
